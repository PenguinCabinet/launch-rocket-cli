# 🚀 launch-rocket-cli
![紹介GIF](img/anime1.gif)

これはロケットを打ち上げるコマンドラインツールです。     
オプションで墜落させることもできます。

## install
```
go install github.com/PenguinCabinet/launch-rocket-cli@latest
```

## usage
### 単にロケットを打ち上げたい場合
```
launch-rocket-cli
```
### 打ち上げた後、ロケットを墜落させたい場合
```
launch-rocket-cli -f
```
### 詳しい使い方
```
launch-rocket-cli -h
```
