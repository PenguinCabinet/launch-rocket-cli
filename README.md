# 🚀 launch-rocket-cli
![紹介GIF](img/anime1.gif)

これはロケットを打ち上げるコマンドラインツールです。     
オプションで墜落させることもできます。

This is a command-line tool to launch the rocket into space.     
It also has an option to crash them.

## install
```
go install github.com/PenguinCabinet/launch-rocket-cli@latest
```

## usage
### Launch the rocket
```
launch-rocket-cli
```
### Crash a rocket
```
launch-rocket-cli -f
```
### Detailed instructions
```
launch-rocket-cli -h
```
